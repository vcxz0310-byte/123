# 로또 번호 랜덤 추천 웹사이트

브라우저에서 **1~45 중복 없이 6개 + 보너스 1개** 로또 번호를 랜덤으로 추천해주는 간단한 정적 웹페이지입니다.

## 실행 방법

- `index.html`을 더블클릭해서 브라우저로 열면 됩니다.

## 기능

- 한 번에 **5세트** 추천
- 번호는 자동 오름차순 정렬(보너스 포함)
- 결과 복사(클립보드)
- 최근 추천 기록 자동 저장(브라우저 LocalStorage)
- 토스트 알림(추천/복사/삭제)
- 단축키: Enter(추천), Ctrl+C(복사), Esc(알림 닫기)
