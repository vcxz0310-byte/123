# 뉴스 요약 챗봇

키워드를 입력하면 Google 뉴스에서 관련 기사를 최대 10개 찾아 재미나이(Gemini) AI로 요약하고, 뉴스에 대해 대화할 수 있는 웹 애플리케이션입니다.

## 주요 기능

1. **🔑 API 키 설정 및 검증**
   - 재미나이(Gemini) API 키를 입력하고 즉시 검증
   - 검증 결과를 색상과 아이콘으로 명확하게 표시
   - API 키는 평문으로 표시 (비밀번호 숨김 없음)

2. **🔍 뉴스 검색**
   - Google 뉴스 RSS에서 키워드로 최대 10개 기사 수집
   - 네트워크 오류 시 구체적인 오류 메시지 표시
   - 수집된 뉴스 목록과 상세 정보 표시

3. **📝 AI 요약**
   - 재미나이 AI를 사용하여 수집한 뉴스 전체를 요약
   - 3-5문장으로 핵심 내용 요약

4. **💬 뉴스 대화**
   - 수집한 뉴스에 대해 질문하고 답변 받기
   - 재미나이 AI가 뉴스 내용을 바탕으로 답변
   - 대화 창이 명확하게 표시됨

5. **💾 뉴스 저장**
   - 검색한 키워드와 뉴스 기사들을 JSON 파일로 저장
   - 저장된 뉴스 조회 기능

## 설치 방법

1. Python 3.9 이상이 설치되어 있는지 확인합니다.

2. 필요한 라이브러리를 설치합니다:
```bash
pip install -r requirements.txt
```

## 실행 방법

### 방법 1: start.bat 사용 (권장)

1. `start.bat` 파일을 더블클릭합니다.
2. 자동으로 Flask 서버가 시작되고 브라우저가 열립니다.
3. `http://localhost:5000`에서 애플리케이션을 사용할 수 있습니다.

### 방법 2: 수동 실행

1. PowerShell 또는 명령 프롬프트를 엽니다.
2. 프로젝트 폴더로 이동합니다:
```bash
cd "C:\Users\admin\Desktop\chatbot"
```

3. Flask 서버를 실행합니다:
```bash
python news_chatbot_web.py
```

4. 브라우저에서 `http://localhost:5000`을 엽니다.

## 사용 방법

1. **API 키 설정** (최초 1회)
   - 상단 "API 키 입력"란에 재미나이 API 키를 입력합니다.
   - "검증 및 저장" 버튼을 클릭합니다.
   - 검증 결과가 표시됩니다.

2. **뉴스 검색**
   - "뉴스 검색" 섹션에서 키워드를 입력합니다.
   - "뉴스 검색" 버튼을 클릭합니다.
   - 수집된 뉴스가 왼쪽 "뉴스 목록"과 상단에 표시됩니다.

3. **AI 요약 생성**
   - 뉴스 검색 후 "AI 요약 생성" 기능을 사용할 수 있습니다.
   - (현재 버전에서는 JavaScript에서 generateSummary() 함수 호출 필요)

4. **뉴스 대화**
   - 오른쪽 "뉴스 대화" 창에서 질문을 입력합니다.
   - "전송" 버튼을 클릭합니다.
   - 재미나이 AI가 뉴스 내용을 바탕으로 답변합니다.

5. **뉴스 저장**
   - 뉴스 검색 후 저장 기능을 사용할 수 있습니다.
   - "저장된 뉴스" 섹션에서 이전에 저장한 뉴스를 확인할 수 있습니다.

## 파일 구조

- `news_chatbot.py` - 핵심 기능 모듈 (뉴스 검색, AI 요약, 대화, 저장)
- `news_chatbot_web.py` - Flask 웹 서버 및 API 엔드포인트
- `index1.html` - HTML 파일 (참고용, Flask 서버를 통해 제공됨)
- `start.bat` - 서버 실행 스크립트
- `test_functions.py` - 기능 검증 테스트 스크립트
- `requirements.txt` - 필요한 Python 라이브러리 목록
- `api_key.json` - 저장된 API 키 (자동 생성)
- `saved_news.json` - 저장된 뉴스 데이터 (자동 생성)

## 주의사항

- 재미나이 API 키는 Google AI Studio에서 발급받을 수 있습니다.
- API 키는 `api_key.json` 파일에 저장되며, 보안에 주의하세요.
- 네트워크 연결이 필요합니다 (Google 뉴스 RSS 및 재미나이 API 접근).

## 문제 해결

- **뉴스 검색이 안 될 때**: 인터넷 연결을 확인하세요.
- **API 키 검증 실패**: API 키가 올바른지 확인하고, 재미나이 API 사용량을 확인하세요.
- **대화 기능이 작동하지 않을 때**: 먼저 뉴스를 검색했는지 확인하세요.
